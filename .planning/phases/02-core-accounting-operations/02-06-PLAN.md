handle_registrar_pago_cuota(tool_input, alegra, db, event_bus, user_id):
  - tool_input: {loanbook_id, numero_cuota, monto, banco, fecha}
  - validate_write_permission("contador", "POST /payments", "alegra")
  - loanbook = await db.loanbook.find_one({"loanbook_id": loanbook_id})
  - Find cuota by numero: next((c for c in loanbook.get("cuotas", []) if c["numero"] == numero_cuota), None)
  - If cuota not found: return error
  - If cuota["estado"] == "pagada": return {"success": False, "error": f"Cuota {numero_cuota} ya pagada el {cuota.get('fecha_pago')}"}
  - POST /payments to Alegra via request_with_verify
  - On success: db.loanbook.update_one({"loanbook_id": loanbook_id, "cuotas.numero": numero_cuota},
                                        {"$set": {"cuotas.$.estado": "pagada", "cuotas.$.fecha_pago": fecha}})
  - event_bus.publish("pago.cuota.registrado", ...)
  - return {"success": True, "payment_id": ..., "message": "..."}
